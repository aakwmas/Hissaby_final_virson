# تطبيق حسابي

تطبيق محاسبة شخصية لإدارة الدخل والمصروفات والأصول والخصوم مع دعم اللغة العربية واتجاه RTL.

## المميزات
- تتبع الدخل والمصروفات
- تتبع الأصول والخصوم
- حساب صافي الثروة
- ميزانية شهرية
- تقارير ورسوم بيانية
